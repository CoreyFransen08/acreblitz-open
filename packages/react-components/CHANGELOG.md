# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-01

### Added

- Initial release of `@acreblitz/react-components`
- `Weather` component for displaying current conditions and hourly forecast
- `WeatherDisplay` component for custom weather layouts
- `WeatherSkeleton` component for loading states
- `useWeather` hook for fetching weather data
- Utility functions for unit conversions
- TypeScript type definitions
- CSS module styles

### Features

- Direct browser fetch to National Weather Service API (api.weather.gov)
- 24-hour grid point caching
- Configurable refresh interval
- Imperial and metric unit support
- Compact mode option
- Custom loading and error components support
